Описание предметной области
Цель:
Целью работы является моделирование метрополитена, то есть реализация перемещения по станциям и веткам метро, с осуществлением пересадок.

Постановка задачи:
Задачей данного проекта является реализация с помощью имитационного моделирования способов составления оптимального пути движения пользователя по метрополитену с подробным описанием дальнего пути, количеством пересадок и затраченным временем.

Модель задачи:
Модель задачи — это ориентированный граф, состоящий из множества вершин и множества ребер, соединяющих вершины. Вершины символизируют станции города, ребра обозначают перегоны между станциями, а каждая станция принадлежит определённой ветке, т.е. совокупность станций и перегонов образуют ветку.

•Класс Station является классом для описания и создания станций метро.
•Класс Line является классом для описания и создания перегонов метро.
•Класс Line Metro является классом для описания веток.
•Класс Metro Map является основным классом, реализующим ключевую задачу в данной работе – составление пути.
